# Zoom out

I don't know this area of code well. Go up a layer of abstraction. Give me a map of all the relevant modules and callers, using the project's domain glossary vocabulary.

**Usage:**
- `/user:zoom-out` — when you're unfamiliar with a section of code or need to understand how it fits into the bigger picture
- Say "zoom out" to trigger this mode on any codebase area

---

**Output:** Produce a readable map of the surrounding system — what calls this code, what this code calls, what concepts from the domain model are at play, and where the boundaries of this module are. Use the vocabulary of the project (names from CLAUDE.md, existing class names, layer names) rather than generic terms.
